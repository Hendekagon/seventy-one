# seventy-one

The number 71 in Clojure

## Obtention

`[com.gfredericks/seventy-one "0.1.0"]`

## Usage

``` clojure
(require '[com.gfredericks.seventy-one :refer [seventy-one]])

(defn business-logic
  [customer-count]
  ;; add 71 to customer-count
  (+ customer-count seventy-one))
```
